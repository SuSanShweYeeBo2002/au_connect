11. Troubleshooting

11.1 Common Issues

‚ùå Can't Sign In

Solution:

  • Verify your email and password are correct
  • Check if you've verified your email address
  • Try resetting your password
  • Ensure you have an active internet connection

‚ùå Images Not Loading

Solution:

  • Check your internet connection
  • Try refreshing the page/screen
  • Clear app cache (on mobile devices)
  • Ensure images are in supported formats (JPG, PNG)

‚ùå Posts Not Appearing

Solution:

  • Pull down to refresh the feed
  • Check if you're connected to the internet
  • Log out and log back in
  • Make sure the post was successfully created (check for error messages)

‚ùå Friend Requests Not Working

Solution:

  • Refresh the friends page
  • Check if you already sent a request to that user
  • Verify the user hasn't blocked you

11.2 Error Messages

Error Meaning Solution
Please fill all required fields Missing information in a form Complete all fields marked as required
Passwords do not match Password and confirmation don't match Retype passwords to match exactly
No changes to update No modifications made to profile Change at least one field before saving
Failed to load profile Server connection issue Check internet and try again
